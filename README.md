# Digital File Carving Recovery Analysis

## Overview
This project is about file carving and data recovery in digital forensics. We used different recovery tools to test how well deleted and fragmented files can be recovered from storage media. The project mainly compares the performance of Foremost, PhotoRec, and Scalpel.

## Project Goals
- Study file carving techniques in digital forensics
- Compare different recovery tools
- Analyze recovery accuracy and performance
- Test recovery under normal and fragmented scenarios

## Dataset
The dataset includes different file types such as:
- JPEG and PNG images
- PDF and DOCX documents
- MP4 and MOV videos

🔗 Kaggle Dataset:
https://www.kaggle.com/datasets/janaalgh/digital-file-carving-recovery-dataset

## Tools Used
- Foremost
- PhotoRec
- Scalpel

## Results
The project compares the recovery tools based on:
- Recovery rate
- Accuracy
- False positives
- Processing time

## Academic Purpose
This repository was created as part of a university digital forensics project to understand and evaluate file carving recovery techniques.
